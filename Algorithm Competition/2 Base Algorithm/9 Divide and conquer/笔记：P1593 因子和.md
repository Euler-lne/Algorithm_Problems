## 📘 题目回顾：P1593 因子和

输入整数 $a, b$，求 $a^b$ 的**所有正因子之和**，结果对 **$9901$** 取模。  
数据范围：$1 \le a \le 5\times 10^7$，$0 \le b \le 5\times 10^7$。

---

## 🔐 第一部分：逆元（模意义下的除法）

### 1. 为什么需要逆元？

本题核心公式（对于质因子 $p$ 及其指数 $E$）：

$$
\text{因子和贡献} = 1 + p + p^2 + \cdots + p^E = \frac{p^{E+1} - 1}{p - 1}
$$

但在**取模运算**中，**不能直接使用普通除法**。  
因为 $(p^{E+1} - 1)$ 取模后**不一定能被 $(p-1)$ 整除**，强行用 `/` 会得到错误结果。

**逆元**就是用来在模意义下合法执行“除法”的工具。

### 2. 什么是逆元？

在模 $MOD$ 下，若 $x$ 与 $MOD$ 互质，则存在唯一的 $y \in [0, MOD-1]$ 使得：

$$
x \cdot y \equiv 1 \pmod{MOD}
$$

这个 $y$ 就称为 $x$ 在模 $MOD$ 下的**乘法逆元**，记作 $\operatorname{inv}(x)$ 或 $x^{-1}$。

有了逆元后：

$$
\frac{a}{x} \pmod{MOD} \equiv a \cdot \operatorname{inv}(x) \pmod{MOD}
$$

即 **除以一个数 = 乘以它的逆元**。

### 3. 如何求逆元？（费马小定理法）

若 $MOD$ 是**质数**（本题中 $9901$ 是质数），根据**费马小定理**：

$$
x^{MOD-1} \equiv 1 \pmod{MOD} \quad (\text{其中 } x \not\equiv 0 \pmod{MOD})
$$

将等式两边同时除以 $x$（等价于乘以 $x^{MOD-2}$）：

$$
x^{MOD-2} \equiv \operatorname{inv}(x) \pmod{MOD}
$$

因此，只需用**快速幂**计算 $x^{MOD-2} \bmod MOD$ 即可得到逆元。

```cpp
const int MOD = 9901;

long long qpow(long long a, long long b) {
    long long res = 1;
    a %= MOD;
    while (b) {
        if (b & 1) res = res * a % MOD;
        a = a * a % MOD;
        b >>= 1;
    }
    return res;
}

long long inv(long long x) {
    return qpow(x, MOD - 2);
}
```

### 4. 特例处理

若 $p \equiv 1 \pmod{MOD}$，则 $p-1 \equiv 0 \pmod{MOD}$，无法求逆元（因为 $0$ 没有逆元）。  
此时等比数列每一项模 $MOD$ 均为 $1$，直接有：

$$
1 + p + p^2 + \cdots + p^E \equiv E + 1 \pmod{MOD}
$$

代码中必须对此特判。

---

### 5. 附：快速幂的数学原理

在上面的 `qpow(a, b)` 中，我们使用了一种称为**快速幂**（又称“二分幂”或“二进制指数法”）的算法，其数学基础如下。

#### 核心思想

计算 $a^b$ 时，若直接将 $a$ 自乘 $b$ 次，时间复杂度为 $O(b)$，对于 $b \le 5\times 10^7$ 会超时。快速幂通过**二进制拆分指数**，将时间复杂度降为 $O(\log b)$。

#### 二进制拆分

任意正整数 $b$ 都可以唯一地表示为二进制形式：
$$
b = \sum_{i=0}^{k} c_i \cdot 2^i, \quad c_i \in \{0,1\}
$$
于是：
$$
a^b = a^{\sum c_i 2^i} = \prod_{i=0}^k a^{c_i 2^i}
$$
由于 $c_i \in \{0,1\}$，我们只需要计算那些 $c_i = 1$ 的项 $a^{2^i}$，并将它们乘起来。

#### 迭代过程

我们可以通过反复平方来高效地得到所有 $a^{2^i}$：
- $a^{2^0} = a^1 = a$
- $a^{2^1} = a^2 = (a^{2^0})^2$
- $a^{2^2} = a^4 = (a^{2^1})^2$
- $a^{2^i} = (a^{2^{i-1}})^2$

同时在每一步检查 $b$ 的二进制最低位（`b & 1`）：
- 若为 $1$，则将当前 $a^{2^i}$ 乘入结果；
- 然后将 $b$ 右移一位（`b >>= 1`），相当于处理下一位。

#### 模运算的兼容

根据模运算的性质：
$$
(x \cdot y) \bmod M = ((x \bmod M) \cdot (y \bmod M)) \bmod M
$$
我们在每一步乘法后立即取模，可以保证中间结果不会溢出，并且最终结果满足要求。

#### 代码实现

把 `b` 想象成一个不断被右移的“二进制纸条”，我们每次都只看最右边的那一位，决定要不要乘入当前累积的权重。也就是把 `b` 二进制分解了。

```cpp
long long qpow(long long a, long long b) {
    long long res = 1;
    a %= MOD;
    while (b) {
        if (b & 1)               // 当前二进制位为 1
            res = res * a % MOD;
        a = a * a % MOD;         // 计算 a^(2^i)
        b >>= 1;                 // 处理下一位
    }
    return res;
}
```

#### 示例

计算 $3^{13} \bmod 100$：
- $13$ 二进制为 $1101_2$（即 $8+4+1$）
- 初始：`res=1, a=3`
- $b=13$：最低位为 $1$ → `res = 1*3 = 3`；`a = 3^2=9`；`b=6`
- $b=6$：最低位为 $0$ → `res` 不变；`a = 9^2=81`；`b=3`
- $b=3$：最低位为 $1$ → `res = 3*81=243 % 100 = 43`；`a = 81^2=6561 % 100 = 61`；`b=1`
- $b=1$：最低位为 $1$ → `res = 43*61=2623 % 100 = 23`；`a = 61^2`；`b=0`
- 结果 $23$，与 $3^{13}=1594323$ 取模 $100$ 一致。

## 🧮 第二部分：质因数分解

### 1. 为什么需要分解 $a$？

题目要求的是 $a^b$ 的因子和。  
若 $a = p_1^{e_1} \cdot p_2^{e_2} \cdots p_k^{e_k}$，则 $a^b = p_1^{e_1 b} \cdot p_2^{e_2 b} \cdots p_k^{e_k b}$。

根据**因子和公式**：

$$
\sigma(a^b) = \prod_{i=1}^k \left( 1 + p_i + p_i^2 + \cdots + p_i^{e_i b} \right)
$$

因此必须先将 $a$ 分解为质因子的乘积。

### 2. 分解代码及原理解析

```cpp
long long ans = 1;
for (int i = 2; i * i <= a; i++) {
    if (a % i == 0) {
        int cnt = 0;
        while (a % i == 0) {
            a /= i;
            cnt++;
        }
        // i 是一个质因子，指数为 cnt
        ans = ans * calc_sum(i, 1LL * cnt * b) % MOD;
    }
}
if (a > 1) {
    // 剩下的 a 本身是一个 > sqrt(原a) 的质因子，指数为 1
    ans = ans * calc_sum(a, 1LL * 1 * b) % MOD;
}
```

#### ✅ 原理说明

- **循环条件** `i * i <= a`：  
  一个合数必定有一个不大于其平方根的质因子。随着不断用 `a /= i` 消去较小的质因子，`a` 会变小，循环边界动态缩小，效率更高。

- **统计指数**：  
  `while (a % i == 0)` 持续用 $i$ 去除 $a$，每次成功都增加指数 `cnt`，直到除不尽为止。这样得到的 $i$ 一定是**质数**（因为所有小于 $i$ 的因子已被除尽，能整除 $a$ 的 $i$ 不可能是合数）。

- **最后的 `if (a > 1)`**：  
  循环结束时，如果 $a > 1$，说明 $a$ 是一个大于 $\sqrt{\text{原 } a}$ 的质数，且指数为 $1$（因为最多只有一个这样的质因子）。

### 3. 时间复杂度

试除法最坏情况 $O(\sqrt{a})$，$a \le 5\times 10^7$ 时约需检查 $7000$ 次循环，完全可接受。

## 🧩 完整代码参考

```cpp
#include <cstdio>

const int MOD = 9901;

long long qpow(long long a, long long b) {
    long long res = 1;
    a %= MOD;
    while (b) {
        if (b & 1) res = res * a % MOD;
        a = a * a % MOD;
        b >>= 1;
    }
    return res;
}

long long inv(long long x) {
    return qpow(x, MOD - 2);
}

// 计算 1 + p + p^2 + ... + p^E  (mod MOD)
long long calc_sum(long long p, long long E) {
    if (p % MOD == 1) {
        return (E + 1) % MOD;
    }
    return (qpow(p, E + 1) - 1 + MOD) % MOD * inv(p - 1) % MOD;
}

int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    if (b == 0 || a == 1) {
        printf("1\n");
        return 0;
    }

    long long ans = 1;
    for (int i = 2; i * i <= a; i++) {
        if (a % i == 0) {
            int cnt = 0;
            while (a % i == 0) {
                a /= i;
                cnt++;
            }
            ans = ans * calc_sum(i, 1LL * cnt * b) % MOD;
        }
    }
    if (a > 1) {
        ans = ans * calc_sum(a, 1LL * 1 * b) % MOD;
    }
    printf("%lld\n", ans);
    return 0;
}
```

## ⚠️ 易错点总结

| 问题                      | 后果                  | 解决方案                       |
| ------------------------- | --------------------- | ------------------------------ |
| 直接使用 `/` 做除法       | WA                    | 用逆元将除法转为乘法           |
| 未处理 `p % MOD == 1`     | RE（逆元不存在）或 WA | 特判直接返回 $E+1$             |
| 未考虑 `a = 1` 或 `b = 0` | WA                    | 开头特判输出 $1$               |
| 质因数分解后忘记乘上 $b$  | WA                    | 指数为 $\mathit{cnt} \times b$ |
| 快速幂未取模或溢出        | WA                    | 每一步乘法后都 `% MOD`         |
